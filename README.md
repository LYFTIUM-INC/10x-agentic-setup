### Prerequisites
- **Claude Code** or **Claude Desktop** with MCP support
- **Git** for version control  
- **Node.js 18+** (for npm-based MCPs)
- **Python 3.8+** with pip/uvx (for Python-based MCPs)

### 🔧 Required MCP Servers

#### 1. **Fetch MCP Server** 🌐
```bash

```

